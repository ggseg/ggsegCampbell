Package content:
README_Campbell								Data version, licensing information and list of brain regions, including for each region: region number, digital atlas label, the corresponding original region
'label' folder								Containing individual manually drawn labels, used in atlas digitisation
lh.atlas.gcs & rh.atlas.gcs						Surface-based atlas
lh.fsaverage.Campbell.label.gii & rh.fsaverage.Campbell.label.gii 	Surface-based atlas, applied to fsaverage and converted to .gii surface mesh using mris_convert 
Campbell.nii.gz								Colin27 volume-based atlas
Campbell_NCBI152.nii.gz							NCBI152 volume-based atlas
lh.colortable.txt & rh.colortable.txt					Colortable files for the surface-based atlas
CampbellColorLUT.txt							Lookup table, including ROI number and color code for each region
